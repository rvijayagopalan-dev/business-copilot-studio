# API Copilot — Learning Academy

This Learning Academy provides concise, practical learning materials for the API Copilot product: capabilities, skills, deep dives, best practices, sample prompts, known gaps, and a checklist to onboard engineers and product teams.

Files in this folder:

- `capabilities.md` — capability map and short descriptions.
- `skills.md` — agent skills and role responsibilities.
- `deep-dive.md` — purpose, context, architecture, and implementation notes.
- `best-practices.md` — development and deployment best practices.
- `sample-prompts.md` — curated prompts for common tasks.
- `gaps-checklist.md` — checklist of known gaps and follow-ups.
- `index.html` — quick HTML gateway to the learning pages.

Use these pages when training new team members, writing prompts, or performing reviews.
