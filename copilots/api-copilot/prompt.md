# API Copilot Prompt Examples

## Design Prompts

- "Create Customer API for Banking Domain using BIAN standards. Generate OpenAPI 3.1, data model, and mock server."
- "Design a GraphQL API for an event ticketing system with subscriptions and authorization rules."
- "Generate an AsyncAPI contract for a real-time order processing workflow using Kafka."

## Development Prompts

- "Generate Java Spring Boot CRUD API with clean architecture, DTOs, validation, and error handling."
- "Create a FastAPI service for user profile management with OpenAPI docs and dependency injection."
- "Generate a TypeScript NestJS API module for product catalog management."

## Testing Prompts

- "Create a Postman collection for the Payments API with positive and negative test cases."
- "Generate integration tests for the Orders API using Rest Assured and contract validation."
- "Create a performance test plan for the API gateway with k6 scenarios."

## Security Prompts

- "Review this OpenAPI spec for OWASP Top 10 issues and recommend specific fixes."
- "Generate an API threat model for the customer onboarding API."
- "Validate JWT authentication flow and suggest improvements for token handling."

## Governance and Documentation Prompts

- "Generate API governance rules for versioning, deprecation, and authorizations."
- "Create developer portal content for the Billing API with examples and SDK snippets."
- "Summarize compliance requirements for API data privacy and audit logging."
