# api-copilot

This folder contains the Claude product definition and framework for the Enterprise API Copilot.

## Purpose

API Copilot is a standardized product that enables natural-language-driven API engineering across design, development, testing, security, documentation, governance, and observability.

## Files

- `SKILL.md` — defines the product capabilities, agent roles, and expected outputs.
- `copilot-instructions.md` — instructs how the copilot should behave and what outputs to produce.
- `prompt.md` — captures prompt templates and example user requests.
- `README.md` — explains the product folder and the standard template for future copilots.

## Standard Copilot Folder Pattern

Future copilots should follow this same structure to stay consistent and reusable:

1. `SKILL.md`
2. `copilot-instructions.md`
3. `prompt.md`
4. `README.md`

This creates a repeatable framework for capturing product knowledge, behavior rules, examples, and onboarding guidance.

## Notes

The content in this folder is based on the API Copilot documentation in `docs/api/api-copilot.md` and the API copilot reference model.
