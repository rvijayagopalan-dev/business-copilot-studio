# copilot-template

This folder is a reusable template for creating new Claude copilot products.

## Purpose

Use this folder as the starting point for any new copilot product in this workspace.

## Standard File Set

- `SKILL.md`
- `copilot-instructions.md`
- `prompt.md`
- `README.md`

## How to Use

1. Copy this folder and rename it for the new product.
2. Update `SKILL.md` with the product's capabilities and domain knowledge.
3. Update `copilot-instructions.md` with product-specific behavior and output rules.
4. Update `prompt.md` with example prompts that reflect the new copilot's use cases.
5. Update `README.md` with folder intent and onboarding details.

## Product Template Goals

- Standardize new copilot creation
- Preserve consistent behavior across copilots
- Provide a ready-to-use framework for domain-specific products
