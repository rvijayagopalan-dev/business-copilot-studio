# Copilot Template Prompt Examples

## General Prompts

- "Help me create a new copilot for [domain]."
- "Generate product documentation and behavior rules for a Claude assistant that handles [task]."
- "Define the capabilities, output style, and prompt examples for a new copilot product."

## Domain-Specific Prompts

- "Create a prompt library for a financial services copilot."
- "Generate example prompts for a developer copilot that writes infrastructure as code."
- "Define a template for a HR copilot that manages employee onboarding workflows."

## Instructions Prompts

- "List the interaction rules for this copilot template."
- "Summarize what output format this copilot should produce."
- "Create a response style guide for the copilot product."
