# Dev Copilot Demo Prompt Examples

## Example Prompts

- "Build a small HTML/CSS/JavaScript calculator demo and include Agile user stories."
- "Generate a calculator UI with buttons for addition, subtraction, multiplication, and division."
- "Create acceptance criteria and test cases for a simple web calculator."
- "Provide a sprint task list for developing an HTML calculator using Agile practices."

## Prompt Templates

- "Generate a demo web calculator with the following features: [feature list]. Include a user story, acceptance criteria, and test cases."
- "Create a simple front-end calculator page with HTML, CSS, and JavaScript. Use a clean layout and basic styling."
- "Write Agile SDLC artifacts for a small calculator feature: user stories, acceptance criteria, tasks, and a test plan."
